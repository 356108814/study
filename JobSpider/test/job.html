职位诱惑 : 地铁上盖，弹性制上班，双休，五险一金

2016-08-18  发布于拉勾网

职位描述

1. 参与高并发的后台处理系统的需求整理和开发工作

2. 负责系统性能优化和改进

3. 运用在分布式系统，网络设计及大型存储系统的知识，构建能够满足商业需求的平台，系统及网络基础结构。

4. 编写，分析并优化新的算法或数据结构并证明比现有方案更优。


岗位要求：

1. 熟悉Python3，熟悉Java、PHP其中一门语言，并具备良好的编程风格，以及能够撰写良好的技术文档；

2. 深入理解和使用XML, JSON and RESTful Web Services

3. 熟悉消息组件，NoSQL、PostgreSQL数据库的使用；

4. 熟悉Linux环境，具备3年以上的linux环境开发经验；

5. 具备系统架构设计能力，逻辑思维清晰，有独立设计开发项目经验者优先；

6. 有较强的分析问题与独立解决问题的能力，良好的沟通能力及团队合作精神；

7. 有分布式系统设计经验者优先。

8. 有机器学习，数据挖掘，自然语言处理经验者优先
